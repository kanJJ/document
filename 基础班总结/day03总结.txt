3,流程控制语句(掌握)
	(1)if语句
		A:if语句的格式1

			a:格式
				if(关系表达式) {
					语句体;
				}
			b:执行流程
				执行关系表达式,看结果是true还是false
				如果是true,执行语句体,if语句结束
				如果是false,结束if语句
		B:if语句的注意事项
			a:关系表达式不管是简单还是赋值,结果都是布尔类型
			b:有左大括号就没有分号,有分号就没左大括号
			c:语句体如果是一条语句的话,大括号可以省略,但是不建议
		C:if语句的格式2
			a:格式:
				if(关系表达式) {
					语句体1;
				} else {
					语句体2;
				}
			b:执行流程
				执行关系表达式,看结果是true还是false
				如果是true,执行语句体1,if语句结束
				如果是false,执行语句体2,if语句结束
		D:if语句的格式2和三元表达式的区别
			能用三元表达式写都可以用if语句改写
			能用if语句写,如果语句体是输出语句,就不能用三元运算符改写
		E:if语句的格式3
			a:格式
				if(关系表达式1) {
					语句体1;
				} else if(关系表达式2) {
					语句体2;
				}
				...
				else {
					语句体n+1;
				}
			b:执行流程
				执行关系表达式1,看结果是true还是false
				如果是true,执行语句体1,结束if语句
				如果是false,看关系表达式2的结果是true还是false
				如果是true,执行语句体2,结束if语句
				如果是false...
				如果都不满足,执行语句体n+1
		F:练习题
			求两个整数的最大值
			求三个整数的最大值
			求两个整数是否相等
			求两个数之和
			求三个数之和

	(2)switch语句
		A:格式
			switch(表达式) {
				case 值1:
					语句体1;
					break;
				case 值2:
					语句体2;
					break;
				...
				default:
					语句体n+1;
					break;
			}
		B:格式说明
			a:表达式:byte,char,short,int,枚举(JDK1.5),String(字符串)(JDK1.7)
			b:case:case后的值是用来和表达式的结果配置的,如果满足执行对应的语句体
			c:break:用来结束switch语句
			d:default:如果都不满足,执行default后的语句体
		C:执行流程
			用表达式的结果和case后的值匹配,匹配上就执行对应的语句体.
		D:注意事项
			a:case后的值只能是常量,不能是变量
			b:多个case后的值不能相同
			c:break可以省略吗?
				可以,但是不建议.会造成case穿透
			d:default必须放在最后吗?
				不是.可以放在任意位置.建议放在最后
			e:switch语句的结束条件
				遇到break
				执行到末尾
		E:练习题
			给定一个值,输出对应星期几

			int x = 2;
			int y = 3;
			switch(x){
			    default:
				y++;
				break;
			    case 3:
				y++;
			    case 4:
				y++;
			}
			System.out.println("y="+y);


			int x = 2;

			switch(x){
			    default:
				y++;
			    case 3:
				y++;
			    case 4:
				y++;
			}
			System.out.println("y="+y);
	(3)if语句和switch的区别
		使用场景
			if语句适用于范围值
			switch适用于固定值
